Atividade 1 - Cálculo da Média de Aluno (If)

Desenvolva uma aplicação web que, a partir de 3 notas de um aluno, informadas pelo usuário, calcule a sua média e de acordo com a mesma emita a situação do aluno:
Aprovado: média >= 7
Recuperação: 5 <= média < 7
Reprovado: média < 5 

<form name="frmnotas" onsubmit="procFormulario(); return false;">
    Nota 01: <input type="text" id="num1"><br>
    Nota 02: <input type="text" id="num2"><br>
    Nota 03: <input type="text" id="num3"><br>
    <button type="submit">Enviar</button>
</form>

<div id="resultado"></div>

<script>
function procFormulario(){
    let n1 = parseFloat(document.getElementById('num1').value);
    let n2 = parseFloat(document.getElementById('num2').value);
    let n3 = parseFloat(document.getElementById('num3').value);
    let media = (n1+n2+n3)/3;
    let situacao = "";

    if(media >= 7){
        situacao = "Aprovado";
    } else if(media >=5 && media < 7){
        situacao = "Recuperação";
    } else {
        situacao = "Reprovado";
    }
    document.getElementById('resultado').innerHTML = "Média: "+media.toFixed(2)+"<br> Situação: "+situacao;
}
</script>

Atividade 2 - Cálculo do Fatorial (While)

Desenvolva uma aplicação web que, a partir de um número fornecido pelo usuário, calcule o seu fatorial utilizando a estrutura de repetição while. 

<form onsubmit="calcularFatorial(); return false;">
    Número: <input type="text" id="num"><br>
    <button type="submit">Calcular</button>
</form>

<div id="resultado2"></div>

<script>
function calcularFatorial(){
    let n = parseInt(document.getElementById('num').value);
    let resultado = 1;
    let i = 1;
    while(i <= n){
        resultado *= i;
        i++;
    }
    document.getElementById('resultado2').innerHTML = "Fatorial de "+n+" é "+resultado;
}
</script>
